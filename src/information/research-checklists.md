# Stargazing Guide — Research Checklists

## Annual Checklist (Do Every December)

**Time required:** ~2 hours once per year
**When:** First week of December (for upcoming year)

### 1. Moon Phase Data (5 minutes)

**Source:** `https://craigchamberlain.github.io/moon-data/api/moon-phase-data/YEAR/`

- [ ] Download moon phase JSON for upcoming year
- [ ] Extract into monthly tables:
  - New Moon dates
  - Full Moon dates + traditional names
  - First/Last Quarter dates
- [ ] Calculate dark sky windows (New Moon ± 5 days) for each month

### 2. Meteor Shower / Moon Interference (30 minutes)

For each major shower, cross-reference peak date with moon phase:

| Shower | Peak | Moon Phase | Interference Rating |
|--------|------|------------|---------------------|
| Quadrantids | Jan 3-4 | | |
| Lyrids | Apr 22-23 | | |
| Eta Aquariids | May 5-6 | | |
| Delta Aquariids | Jul 30-31 | | |
| Perseids | Aug 12-13 | | |
| Orionids | Oct 21-22 | | |
| Leonids | Nov 17-18 | | |
| Geminids | Dec 13-14 | | |
| Ursids | Dec 22-23 | | |

**Rating guide:**
- **None:** New Moon or thin crescent (<15%)
- **Minimal:** Crescent sets early or rises late (15-30%)
- **Low:** Quarter Moon, sets by midnight or rises after 2am
- **Moderate:** Gibbous, but not overhead during best viewing hours
- **High:** Near-full Moon up most of the night
- **Severe:** Full Moon near peak date

### 3. Solar & Lunar Eclipses (15 minutes)

**Source:** `https://www.timeanddate.com/eclipse/`

- [ ] Check for solar eclipses visible from Europe
- [ ] Check for lunar eclipses visible from Europe
- [ ] Note dates, times, visibility percentages

### 4. Planet Oppositions & Events (15 minutes)

**Source:** `https://in-the-sky.org/` or `https://www.timeanddate.com/astronomy/`

- [ ] Mars opposition date (if applicable — ~every 26 months)
- [ ] Jupiter opposition date
- [ ] Saturn opposition date
- [ ] Notable planetary conjunctions
- [ ] Mercury/Venus greatest elongations (optional)

### 5. Supermoons (5 minutes)

**Source:** `https://www.timeanddate.com/moon/phases/`

- [ ] List Full Moons that qualify as Supermoons (perigee within 24h of Full)

### 6. Comets (10 minutes)

**Sources:** 
- `https://www.aerith.net/comet/weekly/current.html`
- `https://heavens-above.com/Comets.aspx`

- [ ] Check for any bright comets predicted for upcoming year
- [ ] Note dates, peak brightness, visibility windows

### 7. Compile Annual Summary (30 minutes)

Create a quick-reference annual calendar:

```
| Month | Moon Events | Meteor Showers | Planets | Special |
|-------|-------------|----------------|---------|---------|
| Jan   | FM: Jan 7   | Quadrantids ⭐ | ...     | ...     |
| Feb   | ...         | ...            | ...     | ...     |
```

## Monthly Checklist (Do Start of Each Month)

**Time required:** 15-20 minutes
**When:** Last few days of previous month or first day of month

### 1. Confirm Pre-Calculated Data (2 minutes)

- [ ] Moon phases correct (from annual data)
- [ ] Meteor shower interference notes accurate
- [ ] Any special events this month (from annual calendar)

### 2. Planet Visibility Check (10 minutes)

**Tool:** Stellarium Web or desktop Stellarium

For each planet, determine:

| Planet | Visible? | Morning/Evening | Constellation | Brightness | Notes |
|--------|----------|-----------------|---------------|------------|-------|
| Mercury | Y/N | M / E | | | |
| Venus | Y/N | M / E | | | |
| Mars | Y/N | M / E / All Night | | | |
| Jupiter | Y/N | M / E / All Night | | | |
| Saturn | Y/N | M / E / All Night | | | |

**How to check in Stellarium:**
1. Set date to 15th of the month, 22:00 local time
2. Look for each planet
3. Note constellation and approximate brightness
4. Check if visible at dawn (06:00) for morning planets

### 3. Comet Update (2 minutes)

- [ ] Check if any new bright comets announced
- [ ] Check if any predicted comets have updated brightness estimates

### 4. Select Featured Objects (5 minutes)

From master DSO list, confirm selection:
- [ ] 2-3 easy (naked eye / binoculars)
- [ ] 2-3 moderate (small telescope)
- [ ] 1-2 challenging (optional)
- [ ] Objects match "best month" for visibility?

### 5. Write Month-Specific Content

- [ ] "At a Glance" summary
- [ ] Planet descriptions with current positions
- [ ] Any late-breaking news or events
- [ ] Best nights recommendations

### 6. Generate/Update Stellarium Links

- [ ] Verify links work
- [ ] Update date parameters if needed
- [ ] Add any new objects to quick links

## Object Selection Guide by Month

Quick reference for which DSOs are optimal each month:

### December-January (Winter)
**Constellation focus:** Orion, Taurus, Gemini, Auriga
**Optimal objects:**
- M42 (Orion Nebula) ⭐
- M45 (Pleiades) ⭐
- M1 (Crab Nebula)
- M35 (Gemini cluster)
- M78 (Orion reflection nebula)
- Horsehead Nebula (advanced)
- Double Cluster
- M31 (still good early evening)

### February-March (Late Winter/Early Spring)
**Constellation focus:** Gemini, Cancer, Leo rising
**Optimal objects:**
- M35 ⭐
- M44 (Beehive Cluster) ⭐
- Rosette Nebula
- NGC 2392 (Eskimo Nebula)
- M41, M46, M47 (Puppis clusters)
- Leo Triplet (M65, M66, NGC 3628)

### April-May (Spring)
**Constellation focus:** Leo, Virgo, Coma Berenices
**Optimal objects:**
- M81/M82 (Bode's Galaxies) ⭐
- M51 (Whirlpool Galaxy) ⭐
- Virgo Galaxy Cluster
- M104 (Sombrero Galaxy)
- M3 (globular cluster)

### June-July (Summer)
**Constellation focus:** Scorpius, Sagittarius, Summer Triangle
**Optimal objects:**
- Milky Way core! ⭐
- M8 (Lagoon Nebula) ⭐
- M20 (Trifid Nebula)
- M16 (Eagle Nebula)
- M17 (Swan Nebula)
- M13 (Hercules Cluster) ⭐
- M57 (Ring Nebula)
- Albireo (double star)

### August-September (Late Summer)
**Constellation focus:** Cygnus, Lyra, Sagittarius setting
**Optimal objects:**
- M31 (Andromeda rising) ⭐
- M27 (Dumbbell Nebula) ⭐
- NGC 7000 (North America Nebula)
- Veil Nebula
- M11 (Wild Duck Cluster)
- Milky Way still visible

### October-November (Autumn)
**Constellation focus:** Andromeda, Pegasus, Perseus
**Optimal objects:**
- M31 (Andromeda at best) ⭐
- M33 (Triangulum Galaxy)
- Double Cluster ⭐
- NGC 7331 + Stephan's Quintet
- M52, NGC 7789 (Cassiopeia clusters)
- California Nebula (advanced)
